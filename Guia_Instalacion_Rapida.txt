===============================================
        NFS-App - Instalación Rápida
===============================================

Requisitos de Software -> Instalar estos componentes de Software antes de [Instalar la Aplicación]
---------------------
- Sistema: OpenSUSE 15.6 o superior
- Java 17 -> Ejecutar el comando -> 
	sudo zypper install java-17-openjdk-devel

- Servidor NFS -> Ejecutar el comando -> 
	sudo zypper install nfs-kernel-server

- Permisos de administrador (sudo) -> Ejecutar el comando -> 
	sudo zypper install polkit

- Para el desarrollo de la aplicación se utilizaron los entornos de Visual Studio Code e IntellijIdea, pero para crear el archivo ejecutable solo es necesario seguir los pasos de la parte inferior.

Instalación
-----------
1. Abra una terminal en la carpeta raíz del proyecto.
2. Haga ejecutable el script de instalación:
   chmod +x install.sh

3. Ejecute el instalador:
   sudo ./install.sh

> Esto instalará dependencias, compilará y preparará la app.
> Si es la primera vez que se ejecuta el script en promedio tarda 2 minutos en finalizar

Ejecución de la aplicación
-------------------------
La aplicación se ejecuta exclusivamente desde terminal.

Opción 1: Usar el script generado:
    ./App_NFS_Suse.sh

Opción 2: Ejecutar directamente el JAR:
    sudo java -jar build/libs/nfs-app-1.0.0.jar

Nota: Se solicita contraseña de administrador una sola vez al iniciar.

Uso básico
----------
- Agrega directorios y reglas desde la interfaz.
- Haz clic en "Save & Apply" para guardar los cambios (se te pedirá la contraseña de administrador).

Notas importantes
-----------------
- Se requieren permisos de administrador (sudo).
- La aplicación se ejecuta solo desde terminal.
- Verifica que el usuario tenga permisos de sudo.

Desarrollo y compilación
------------------------
- Para recompilar tras cambios en el código:
    ./gradlew build

- Para generar un nuevo JAR ejecutable:
    ./gradlew shadowJar

Soporte
-------
- Si tiene errores, revise los logs con:
    sudo journalctl -u nfs-server -n 50
